# MathCards

**Интерактивная система флэш-карточек для изучения математических теорем и формул при подготовке к ЕГЭ.**

## О проекте

MathCards — это веб-приложение, которое помогает эффективно запоминать математические теоремы, формулы и алгоритмы с помощью системы интервальных повторений. Проект ориентирован на учащихся, готовящихся к ЕГЭ по математике.

## Основные возможности

### Структура карточек

- Двухуровневая система: краткий просмотр и детальная информация
- Индикаторы статуса изучения и сложности
- Примеры использования в задачах ЕГЭ

### Система изучения

- Фильтрация карточек по сложности и статусу
- Отслеживание прогресса изучения
- История последнего обновления

### Визуальный дизайн

- Поддержка светлой и темной темы
- Адаптивный интерфейс для различных устройств
- Интуитивная навигация

## Содержание

Проект включает карточки по всем основным разделам математики:

- **Алгебра**: уравнения, неравенства, функции
- **Геометрия**: планиметрия, теоремы, свойства фигур
- **Стереометрия**: объемы, сечения, пространственные фигуры
- **Анализ**: производные, интегралы, графики функций

## Цели проекта

1. Создать удобный инструмент для систематического повторения математических формул, теорем и т.п.
2. Сократить время на поиск и проверку теоретического материала
3. Повысить эффективность подготовки к экзаменам
4. Предоставить визуально приятный и функциональный интерфейс для обучения

## Особенности

- Чистый и минималистичный дизайн
- Быстрая работа без перезагрузок страниц
- Легкость в добавлении новых карточек

Проект предназначен для всех, кто изучает математику — от школьников до студентов и преподавателей.
